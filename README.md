# Weather Dashboard

A simple and modern weather dashboard built using React and Vite.

## Features

- Search for current weather and 5-day forecast of multiple cities
- Display temperature, humidity, wind speed, and feels-like temperature
- Weather condition icons included
- Loading spinner and error handling
- Responsive design for mobile and desktop
- Dark/light mode toggle

## Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Start the development server with `npm run dev`

## API

This project uses the OpenWeatherMap API to fetch weather data.
